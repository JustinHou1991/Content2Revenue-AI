## 🎯 一、项目概述

### 1.1 项目背景与问题

**核心问题**：内容团队不知道自己做的视频带来了什么质量的线索，销售团队也不知道哪些内容特征在驱动高价值客户。

**用户痛点**：
- 花3小时写的文案，发出去却没人看
- 客户咨询很多，但转化率低得可怜
- 不知道哪条内容能爆，全凭感觉
- 客户需求分散，难以精准匹配产品

### 1.2 解决方案

Content2Revenue-AI 用AI分别分析抖音脚本（内容特征）和销售线索（客户画像），在语义空间中进行匹配，生成数据驱动的内容策略建议。

**核心价值**：不是传统的归因分析（需要数据有关联），而是**智能推荐** — 基于语义理解，告诉内容团队"应该做什么样的内容来吸引高价值线索"。

## 🤖 二、AI Agent使用情况

### 2.1 Agent在工作流中的角色

Content2Revenue-AI 是一个典型的 **AI Agent工作流系统**，包含4个核心Agent：

```
┌─────────────────────────────────────────────────────────────┐
│                    Multi-Agent Orchestration                │
│                                                              │
│  ┌──────────────┐    ┌──────────────┐    ┌──────────────┐ │
│  │ ContentAgent │───▶│  LeadAgent   │───▶│ MatchAgent   │ │
│  │ (内容分析)    │    │ (线索分析)    │    │ (智能匹配)    │ │
│  └──────────────┘    └──────────────┘    └──────────────┘ │
│          │                                        │        │
│          └────────────────┬───────────────────────┘        │
│                           ▼                                │
│                  ┌──────────────┐                          │
│                  │StrategyAgent │                          │
│                  │ (策略顾问)   │                          │
│                  └──────────────┘                          │
└─────────────────────────────────────────────────────────────┘
```

### 2.2 Agent详细说明

#### Agent 1: ContentAnalyzer Agent（内容分析Agent）

**职责**：分析抖音/B站等视频脚本，提取内容特征

**使用场景**：
- 输入：视频脚本文本
- 核心能力：Hook类型识别、情感曲线分析、CTA评估、受众匹配
- 输出：结构化内容画像（15+维度特征）

**Prompt设计要点**：
```
你是一位资深的短视频内容策略分析师，擅长拆解抖音/B端内容营销视频的底层逻辑。

分析维度包括：
1. Hook分析（类型、强度、关键词）
2. 情感分析（基调、变化曲线）
3. 叙事结构（PAS/AIDA/STAR等）
4. CTA分析（类型、清晰度）
5. 受众与话题（标签、目标人群、内容类型、转化阶段）
6. 综合评估（评分、卖点、改进建议）

请严格按照要求的JSON格式输出。
```

#### Agent 2: LeadAnalyzer Agent（线索分析Agent）

**职责**：从销售对话记录中提取客户画像

**使用场景**：
- 输入：客户对话文本、需求描述
- 核心能力：需求识别、痛点提取、购买阶段判断、意向度评估
- 输出：结构化客户画像

#### Agent 3: MatchEngine Agent（匹配Agent）

**职责**：评估内容与线索的语义匹配度

**使用场景**：
- 输入：内容特征 + 客户画像
- 核心能力：5维度匹配度计算（受众匹配/痛点相关/阶段对齐/CTA适当/情感共鸣）
- 输出：匹配分数 + 优化建议

#### Agent 4: StrategyAdvisor Agent（策略顾问Agent）

**职责**：生成数据驱动的营销策略建议

**使用场景**：
- 输入：匹配分析结果
- 核心能力：内容优化建议、分发策略规划、A/B测试方案
- 输出：可执行的策略文档

### 2.3 Agent协作流程

```
用户输入脚本 → ContentAgent分析 → 获得内容特征
     ↓
用户上传线索 → LeadAgent分析 → 获得客户画像
     ↓
     合并
     ↓
MatchAgent匹配 → 计算5维度匹配度 → 生成优化建议
     ↓
StrategyAgent建议 → 输出可执行策略
```
