## 一、问题分析

### 1.1 内容端的问题

我们分析了1000+条抖音教育类视频，发现：

| 问题 | 占比 | 影响 |
|------|------|------|
| Hook太弱 | 67% | 前3秒流失率高 |
| CTA不清晰 | 54% | 用户不知道下一步做什么 |
| 情绪曲线太平 | 48% | 完播率低 |
| 卖点不突出 | 42% | 转化意愿弱 |

### 1.2 线索端的问题

从私信和评论区来的线索：
- 70%只是随便问问，没有真实需求
- 20%有意向但处于认知期，需要培育
- 只有10%是决策期的高意向客户

**核心问题：内容和线索不匹配**

## 二、解决方案

我设计了一套"Content2Revenue-AI"系统，核心逻辑是：

```
内容分析 → 线索评估 → 智能匹配 → 策略建议
```

### 2.1 内容分析模块

用AI分析视频脚本的12个维度：

```python
# 核心分析维度
analysis = {
    "hook_type": "痛点反问型",      # Hook类型
    "hook_strength": 8.5,           # Hook强度 0-10
    "emotion_tone": "焦虑→希望",     # 情绪基调
    "cta_type": "评论区互动型",      # CTA类型
    "cta_clarity": 7.0,             # CTA清晰度
    "target_audience": "25-35岁家长", # 目标受众
    "content_score": 7.5,           # 综合评分
}
```

**Prompt设计要点：**
- 给AI明确的评分标准（什么是8分的Hook）
- 要求输出结构化JSON，方便后续处理
- 加入行业基准对比（超过80%同类内容）

### 2.2 线索评估模块

从客户互动记录中提取关键信息：

```python
lead_profile = {
    "industry": "教育培训",
    "buying_stage": "考虑期",  # 认知期/考虑期/决策期
    "intent_level": 7,         # 意向度 0-10
    "pain_points": ["招生成本高", "传统方式效果差"],
    "urgency": "中",
}
```

**关键洞察：** 如果线索处于"决策期"，但你的CTA是"评论区扣1"，这就是错配。

### 2.3 智能匹配引擎

5维度评分匹配度：

| 维度 | 权重 | 说明 |
|------|------|------|
| 目标受众匹配 | 25% | 内容受众 vs 线索画像 |
| 痛点相关性 | 25% | 内容卖点 vs 线索痛点 |
| 阶段对齐 | 20% | 内容阶段 vs 购买阶段 |
| CTA适当性 | 20% | CTA类型 vs 紧迫程度 |
| 情感共鸣 | 10% | 情绪基调 vs 期望 |

**匹配度解读：**
- 8-10分：完美匹配，立即跟进
- 6-8分：良好匹配，可以推荐
- 4-6分：一般匹配，有优化空间
- <4分：不匹配，需要调整

### 2.4 策略建议模块

不只是告诉用户"匹配度低"，还要告诉"怎么改"：

```json
{
  "gap_analysis": {
    "weakest_dimension": "cta_appropriateness",
    "gap_reason": "内容使用'评论区扣字'CTA，但线索处于决策期",
    "improvement_suggestion": "将CTA改为'点击链接预约1对1咨询'"
  }
}
```

### 3.2 核心代码

**内容分析器：**

```python
class ContentAnalyzer:
    def analyze(self, script_text: str) -> dict:
        prompt = f"""
        分析以下抖音脚本，提取结构化特征：
        
        脚本：{script_text}
        
        请按以下JSON格式输出：
        {{
            "hook_type": "...",
            "hook_strength": 0-10,
            "emotion_tone": "...",
            "cta_type": "...",
            "content_score": 0-10
        }}
        """
        
        response = self.llm.chat([
            {"role": "system", "content": "你是资深内容营销专家"},
            {"role": "user", "content": prompt}
        ])
        
        return json.loads(response)
```

**匹配引擎：**

```python
class MatchEngine:
    def match(self, content: dict, lead: dict) -> dict:
        prompt = f"""
        评估以下内容与线索的匹配度：
        
        内容特征：{content}
        线索画像：{lead}
        
        请输出：
        1. 综合匹配度（0-10分）
        2. 各维度评分
        3. 最弱维度及改进建议
        """
        
        return self.llm.chat_json(prompt)
```

### 4.2 关键优化点

1. **Hook优化**：从"大家好今天分享..."改为"90%的家长都在犯这个错..."
